The "data" folder is used to store data downloaded from the sites of the "Great National Debate" (Great National Debate)
and "Base officielle des codes postaux" (Official postal code database).

Three data files are downloaded from the python file "./download/downloading.py":

- data/contribution_raw.csv
- data/survey_raw.csv
- data/zipcode_raw.csv

And two backup files:

- data/backup_url.csv
- data/backup_url_zipcode.csv
