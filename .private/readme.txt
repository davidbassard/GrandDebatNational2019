In the folder "./.private/" create a JSON file named "mysql_settings.json" containing the login credentials to 
the MySQL database.

The file "./.private/mysql_settings.json" is structured as follows:

{
	"myhost": "[host]",
	"myuser": "[user]",
	"mypassword": "[password]"
}
